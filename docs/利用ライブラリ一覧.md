## 3. Go 推移的依存（Wails が内部利用）

Wails v2 が利用する推移的依存（`go.mod` の `// indirect` 行）はすべて寛容ライセンス（MIT / BSD 系）。直接コードを呼び出していないため個別掲載は省略。

## 4. ライセンス遵守メモ

- **共通義務**: MIT / BSD / ISC / Apache-2.0 はいずれも、著作権表示とライセンス文の**同梱**が必要。配布物には `THIRD_PARTY_LICENSES.md`（リポジトリ直下）を同梱する。
- **Apache-2.0（DOMPurify）**: `NOTICE` ファイルなし。Apache-2.0 ライセンス文と著作権表示の同梱で義務を果たす。MPL-2.0 側は改変しなければソース開示義務は実質発生しない。
- **フォント（SIL OFL 1.1）**: フォントと共に OFL ライセンス文・著作権表示を同梱。予約フォント名（"Noto"）を改変版に使わない。フォント単体での有償販売は不可（アプリ同梱配布は可）。
- **コピーレフト（GPL 等）の依存は無し**。クローズドソースのまま配布可能。
- **推移的依存**: markdown-it（`linkify-it`/`uc.micro` 等）、mermaid（`d3` 等）にも依存があるが、いずれも寛容ライセンス。
